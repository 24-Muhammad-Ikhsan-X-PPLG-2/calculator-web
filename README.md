## Deskripsi

**Calculator Web** adalah aplikasi kalkulator sederhana berbasis **HTML, CSS, dan JavaScript native**.  
Proyek ini dibuat untuk mempelajari DOM manipulation, event handling, dan dasar-dasar JavaScript.

Aplikasi ini memiliki tampilan sederhana namun responsif dan mudah digunakan.

## Cara Menggunakan

1. Clone repository ini:

```bash
git clone https://github.com/24-Muhammad-Ikhsan-X-PPLG-2/calculator-web.git
```

2. Masuk ke directory project

```bash
cd calculator-web
```

3. Buka file **index.html** di browser kamu.
